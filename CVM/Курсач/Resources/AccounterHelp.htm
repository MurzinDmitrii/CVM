Если Вы менеджер:

Введите в поле поиска ФИО клиента для поиска. Используйте кнопки сортировки для сортировки. Нажмите на кнопку "добавить клиента" чтобы перейти на страницу добавления клиента. Нажмите на кнопку "добавить контракт", чтобы добавить заявку на услугу. Нажмите на кнопку "статистика" чтобы вывести квартальную статистику. Нажмите на кнопку "назад" чтобы вернуться Нажмите на договор ПКМ чтобы вызвать контекстное меню. Нажмите пункт "редактировать" чтобы редактировать информацию о клиенте. Нажмите на пункт "вывести чек" чтобы создать документ с договором

Если Вы врач:

Введите в поле поиска ФИО клиента для поиска. Нажмите на кнопку "прием" чтобы посмотреть, назначен ли Вам сейчас прием Нажмите на клиента ПКМ чтобы вызвать контекстное меню. Нажмите пункт "Перейти к карте" чтобы перейти к карте клиента. Нажмите на пункт "распечатать карту" чтобы создать документ с мед.картой клиента. Нажмите пункт "Отослать карту по email" чтобы отправить клиенту электронное письмо с его мед.картой

На странице мед.карты пациента:

Нажмите ПКМ на сопутствующих заболеваниях чтобы вызвать контекстное меню. Нажмите на пункт "добавить" чтобы добавить сопутствующее заболевание Нажмите ПКМ на назначенных упражнениях чтобы вызвать контекстное меню. Нажмите на пункт "добавить" чтобы добавить упражнение. Нажмите на пункт "удалить" чтобы удалить упражнение. Нажмите на пункт "редактировать" чтобы редактировать упражнение.
